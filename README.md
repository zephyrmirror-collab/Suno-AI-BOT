# Fake Social Media Post Generator

This is a simple web application that allows you to generate fake social media posts with different styles, such as Telegram (Dark and Light) and iOS Messages. You can customize the name, time, message, and avatar of the post, and then download the result as a high-quality PNG image.

## Features

-   **Live Preview:** See your changes in real-time as you type.
-   **Customizable:** Change the name, time, message, and avatar to create the perfect post.
-   **Multiple Styles:** Choose between Telegram (Dark and Light) and iOS Message styles.
-   **High-Quality Export:** Download your creation as a high-resolution PNG image.

## How to Use

1.  **Open `index.html` in your browser.**
2.  **Fill in the fields:**
    *   **Name:** The name of the person "sending" the message.
    *   **Time:** The timestamp for the message.
    *   **Message:** The content of the message.
    *   **Avatar:** Upload an image to be used as the avatar.
3.  **Select a style:** Choose from the dropdown menu to change the look of the post.
4.  **Download your image:** Click the "Download Image" button to save your creation.

## Tech Stack

-   HTML
-   CSS (with Tailwind CSS)
-   Vanilla JavaScript
-   FontAwesome (for icons)
-   html2canvas (for screenshots)
